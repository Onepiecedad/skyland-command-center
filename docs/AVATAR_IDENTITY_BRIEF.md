# Alex Avatar & AI-Influencer Visual Identity Project

## 1. Alex Avatar (för SCC och som din AI-assistent)

### Koncept:
- **Stil:** Futuristisk men vänlig, sci-fi med värme
- **Färger:** Blått/lila (tech) + gult/guld (värme/energi)
- **Element:** Nätverk, data-flöden, hjälpande gestalt
- **Känsla:** Professionell, pålitlig, hjälpsam

### Prompts att generera:

**Alt 1 - Futuristisk Humanoid:**
"A friendly AI assistant avatar, futuristic humanoid robot with glowing blue eyes, minimalist design, soft blue and purple gradients, holographic interface elements, network connections in background, professional yet approachable, clean vector art style, 4k, centered portrait"

**Alt 2 - Neural/Abstract:**
"Abstract AI avatar, geometric neural network pattern forming a face, glowing nodes and connections, blue and cyan colors, data streams flowing, minimalist tech aesthetic, modern digital art, suitable for profile picture, 4k"

**Alt 3 - Sci-Fi Companion:**
"Cute but professional robot assistant avatar, small spherical floating drone with face display, blue LED lights, helping gesture, sci-fi control room background, warm lighting, friendly expression, 3d render style, 4k portrait"

## 2. AI-Influencer Visual Identity (vibe-kodning på svenska)

### Koncept:
- **Namn:** "AI-kompisen" eller något du väljer
- **Stil:** Lättsmält, pedagogisk, men modern och tech
- **Färger:** Lila (kreativitet) + Rosa (energi) + Vit (clarity)
- **Vibe:** "Vibe-coding" = kreativ, visuell, tillgänglig

### Element:
- Kod-snippet visuella element
- AI-hjärna/nätverk som är mjuka och organiska
- Svenska flaggan subtilt integrerad
- Modern, TikTok/Instagram-anpassad estetik

### Prompts:

**Profilbild:**
"Modern AI influencer avatar for Swedish tech education content, friendly character with headphones and holographic code floating around, purple and pink gradient background, neural network patterns, approachable expression, suitable for TikTok and Instagram, vibrant colors, 4k, circular crop suitable for profile"

**Banner/Header:**
"Wide banner for AI education channel, text space in center, floating code snippets and AI brain visualization, Swedish colors subtly integrated (blue/yellow), modern gradient purple to pink, tech but approachable, vibes of creativity and learning, 4k"

**Logo Concept:**
"Minimalist logo for AI education brand, combining code brackets { } with a friendly AI face, Swedish tech aesthetic, clean lines, suitable for small and large sizes, purple and blue color scheme, vector style"
